# Step 20.10 — Wider Polynomial Coefficient Inputs

## What changed
- Widened the coefficient input fields in the polynomial editor.
- Increased input height, padding, and font size so values are visible.
- Added a responsive breakpoint so labels stack above inputs when the available width is limited.

## How to test
1. Open `Abstract Vector Spaces`.
2. Choose `Polynomials`.
3. Check the `Edit Polynomials` area.
4. Verify that the numbers in the coefficient inputs are fully visible for both `p(x)` and `q(x)`.
